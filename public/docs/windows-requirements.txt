Node.js Minimum Requirements for Windows

- Operating System: Windows 7 or later
- Processor: 1 GHz or faster with x64 architecture
- RAM: 2 GB or more
- Disk Space: 1 GB free space
- Additional Software: Windows PowerShell 5.1 or later

Installation Notes:
- Ensure that your Windows installation is up to date with the latest updates.
- You might need administrator rights to install Node.js and its dependencies.
